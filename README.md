# DHCP-Server-Configuration-on-Router

# 🧪 DHCP Configuration Lab

## Overview
Configured a Cisco router as a DHCP server to automatically assign IP addresses to clients on a LAN.

## Skills Demonstrated
- DHCP configuration
- IP address management
- Default gateway assignment
- Network connectivity testing

## Topology
1 Router, 1 Switch, 3 PCs

## Results
Clients successfully received IP addresses automatically and communicated across the network.
